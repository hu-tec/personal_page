# Jinny Park 개인 브랜딩 사이트 — 콘텐츠 & 구조 명세서

## 전체 개요

Jinny Park(진니 박)의 개인 브랜딩 웹사이트. 27년간 언어·교육 분야, 7년간 AI 기술 분야에서 활동한 전문가의 삶과 철학, 경력, 미래 비전을 담은 사이트.

- **분위기**: 우아하고 세련된 프리미엄 톤. 따뜻한 색조(네이비, 골드, 베이지).
- **서체**: 영문 Cormorant Garamond / 한글 Noto Serif KR (세리프 계열, fontWeight 600 기준). Google Fonts에서 import.
- **구조**: React + React Router, 5개 페이지 + 공통 Layout (네비게이션 + 푸터)
- **애니메이션**: 스크롤 시 fade-up 등장 애니메이션 (motion 라이브러리 사용)
- **반응형**: 데스크톱/모바일 대응. 모바일에서는 햄버거 네비게이션.

#### 1) 당신의 강점은 무엇인가요?
  - 우리는 완벽할 수 없어요. 단점을 고치느라 지치기보다는, 당신이 이미 잘하는 것을 더 빛나게 해보세요.
  - 약간의 부족함은 괜찮아요. 그건 곁에 있는 사람들과 연결될 기회가 되기도 하니까요.
  - 심리학자들은 '강점 기반 접근'이 성과와 만족감을 높인다고 말합니다.
  - 누군가의 도움을 받아도 전혀 부끄러운 일이 아니에요. 오히려 그것이 더 건강한 삶의 방식이에요. 시간과 에너지는 한정되어 있어요. 오늘의 행복에 더 집중하세요.
  - 당신의 장점은 세상에 긍정적인 영향력을 줄 수 있는 자산입니다.

#### 2) 하고도 후회, 안 하고도 후회된다면?
  - '해볼 걸 그랬다'는 후회는 오래가요.
  - 시도했을 때의 후회는 한 번이면 끝이지만, 시도하지 않은 후회는 두고두고 마음에 남아요.
  - 행동 심리학에 따르면, 사람은 '하지 않은 일'에 대해 더 오래 후회합니다.
  - 마음이 간다면, 일단 해보는 쪽을 선택해보세요. 삶은 생각보다 짧고, 기회는 늘 다시 오지 않으니까요.
  - 움직이면 경험이 쌓이고, 경험은 결국 지혜가 됩니다.

#### 3) 오늘을 살아보세요.
  - 행복은 미루는 게 아니에요.
  - "더 준비되면", "좀 더 나아지면" 하고 기다리다 보면, 지금의 행복은 놓쳐버려요.
  - 지금 눈에 보이는 작고 평범한 것들 속에 행복이 숨어 있어요.
  - 심리학자 마틴 셀리그먼은 "행복은 외부 조건보다, 내가 지금 어떻게 인식하느냐에 달려 있다."
  - 완벽한 때를 기다리지 마세요.
  - 사랑도, 여행도, 맛있는 음식도 좋은 사람과 지금 느끼는 게 정답이에요.
  - 현재에 집중할 때, 삶은 더 선명하고 깊어집니다.
