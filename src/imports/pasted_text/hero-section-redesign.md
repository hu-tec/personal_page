[Section Name]
Hero Section – Premium Personal Branding

[Goal]
현재 레이아웃을 유지하지 않고, 첨부된 레퍼런스 스타일(밝은 베이지 톤, 이미지 중심 레이아웃)로 완전히 재구성.
텍스트 내용은 그대로 유지하면서 UI/UX와 배치를 고급스럽게 리디자인.

---

[Layout]
- 정렬: 좌측 텍스트 + 우측 인물 이미지
- 구조: 2단 Grid (좌 55% / 우 45%)
- max-width: 1440px
- height: 100vh (풀스크린)
- 여백: 좌우 80px, 상단 120px

---

[Background]
- 전체 배경: 따뜻한 베이지 그라데이션 (#E8DED2 → #CBBBAA)
- 질감: 부드러운 노이즈 or 종이 텍스처 느낌
- 현재 다크 테마 제거

---

[Right Image]
- 인물 이미지를 우측에 크게 배치 (세로 full height 느낌)
- 이미지 스타일:
  - 자연광 느낌
  - 부드러운 shadow
  - 배경과 자연스럽게 블렌딩
- border-radius: 없음 (자연스러운 컷)
- 이미지가 화면 우측 100% 채우도록

---

[Left Content]

1. 상단 통계 카드
- 위치: 좌측 상단
- 스타일:
  - 반투명 브라운 박스 (rgba(60,40,20,0.4))
  - backdrop blur 적용
  - border-radius: 20px
- 내부:
  - 4개 수치 가로 정렬
  - 구분선 vertical divider
- 타이포:
  - 숫자: gold tone (#D6B97B)
  - 설명: small caps

---

2. 메인 타이틀
- 위치: 중앙 좌측
- 텍스트 유지
- 스타일:
  - serif 폰트 (Playfair Display 또는 Noto Serif)
  - 매우 큰 사이즈 (56~72px)
  - line-height 넓게
- 색상:
  - 기본: #FFFFFF 또는 #F5F5F5
  - "AI" 강조: gold color (#D6B97B)

---

3. 서브 텍스트
- 위치: 타이틀 하단
- 스타일:
  - 얇은 sans-serif
  - opacity 70%
  - 줄 간격 넓게

---

4. CTA 버튼
- 텍스트: Explore / Learn More
- 스타일:
  - outline 버튼
  - border: 1px solid #D6B97B
  - hover 시 gold fill

---

[Header]
- 투명 배경 (absolute)
- 스크롤 시:
  → #3A2E23로 변경
- 로고 좌측 / 메뉴 중앙 / SNS 아이콘 우측

---

[Animation]
- 통계 숫자 카운트업 (2~3초 loop)
- 텍스트 fade-in (위에서 아래)
- 이미지 scale-in (1.05 → 1)

---

[Key Style Keywords]
premium / warm / editorial / luxury / calm / personal branding

---

[Important]
- 현재 dark navy UI 제거
- 카드형 이미지 제거 (rounded frame 금지)
- 레이아웃을 단순하게, 이미지 중심으로 변경
- 여백을 충분히 사용해서 고급스럽게 표현
